ACME Currency Converter Manual

Help & Bug Reports

Converting Currencies

Use the spinners to get an exchange rate, input a value into the input textbox to get the conversions between the two currencies.

Saving Currencies

Tap the Save Conversion Button and then enter a valid save name.

Historical Comparisons

Tap the Historical Conversion Button and then enter a valid date.

Refreshing Data

Tap the Refesh Button and Tap OK if you wish to use an API call.

Changing App Settings

When in the Main Activity, Tap on the Menu Bar on the top Right and select the Settings Menu Item. Once in the Settings Menu, you can change the desired Settings.

Loading Saved Conversions

When in the Main Activity, Tap on the Menu Bar on the top Right and select the Saved Conversions Menu Item. Once in the Saved Conversions Activity, select the desired save and click Load Save.


Deleting Saved Conversions

When in the Main Activity, Tap on the Menu Bar on the top Right and select the Saved Conversions Menu Item. Once in the Saved Conversions Activity, select the desired save and click Delete Save.
